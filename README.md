# ReNovo

## 概要

ReNovoは、考えすぎや感情の揺れで行動が止まってしまったときに、
次の一歩を決めるためのシンプルな整理ツールです。<br>
<br>
いくつかの質問に答えるだけで、状況を整理し「今やること」を選べます。

## 背景

日常の中で、以下のような状態に陥ることがあります。

- 考えすぎて手が止まる
- 感情に引っ張られて動けなくなる
- やるべきことは分かっているのに始められない

一般的な対処法として「紙に書き出す」方法がありますが、

- 自由度が高すぎて逆に思考が散らかる
- 何を書けばいいか分からない

といった課題があります。<br>
<br>
自由記述ではなく、あえて制限された入力形式にすることで、
思考の分散を防ぎ、短時間で整理できるようにしています。

## 使い方

1. トップページから開始
2. 現在の状態について簡単な質問に回答
3. 出来事・感情・やりたいことを整理
4. 「今できる行動」を選択
5. 実行後、軽い振り返りや記録
